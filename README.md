# idontwanttomovemyeyes

A speed reader that flashes one word at a time so your eyes stay still. Uses the [Spritz/ORP technique](https://en.wikipedia.org/wiki/Rapid_serial_visual_presentation) — each word is aligned so the "focus letter" always lands in the same spot.

## How to run

No install needed. It's a single HTML file.

1. Clone the repo:
   ```bash
   git clone https://github.com/ShannonLin284/idontwanttomovemyeyes.git
   ```
2. Open `index.html` in any browser — double-click it or drag it into a tab.

That's it.

## How to use

1. Pick a content type: joke, raccoon fact, famous quote, or classic book passage
2. Choose a reading speed (preset levels or drag the slider yourself)
3. Hit **go →** and don't move your eyes

Press `Space` to pause/resume. Click `←` to go back to the menu.

## Speeds

Presets range from 100 wpm (very slow) up to 700+ wpm. The slider lets you set any speed between 80–700 wpm.
